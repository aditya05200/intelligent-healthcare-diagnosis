# Intelligent Healthcare Diagnosis System

## Overview
A web-based application that uses machine learning algorithms to diagnose diseases based on patient symptoms and medical history.

## Technology Stack
- Backend: Java Spring Boot
- Database: MySQL/PostgreSQL
- Machine Learning: TensorFlow/Scikit-Learn
- Containerization: Docker

**Features:**
- Aditya's Report: Displays an overview of the patient's health status, including progress as indicated by a percentage (e.g., 76%).
- Chat with AI Doctor: A feature enabling patients to consult with an AI-based doctor for quick initial diagnostics based on symptoms entered by the patient.
- Lab Tests & Screenings: Access to lab results and scheduled health screenings.
- Checkup Plan: Displays personalized health checkup plans based on the patient's medical history.
- Health Reports: Quick access to the patient's health reports and past diagnoses.
- Personalized Supplements: Provides recommendations for supplements based on the patient's health data, tracking the completion of personalized health plans (e.g., 60% 
  progress).
- Consult Top Doctors: Enables patients to consult with top doctors for further advice or treatment based on the diagnostics.
## Setup Instructions
1. Clone the repository.
2. Navigate to the `backend` directory and set up Spring Boot.
3. Navigate to the `database` directory and set up the database.
4. Use Docker to containerize and run the application.
